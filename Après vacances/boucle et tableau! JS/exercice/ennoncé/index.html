Exo boucle et tableau

1. Réaliser une page web qui permet à l'utilisateur d'encoder des valeurs entiere (avec validation simpl).

Interaction :

  • Afficher les valeurs encodées (Non trier),
  • Afficher la somme des valeurs,
  • Afficher les valerus (min/max),
  • Afficher les valerus trier.


Note: L'utilisateur de la fonction de "sort" n'est pas autorisé. :)



2. Réaliser un site web qui permet d'encoder les scores de 4 joueurs

Sénario :

  • Les joueurs sont anonymes
  • Les scores sont encodés à l'aide de 4 inputs
  • Les scores totaux des joueurs s'actualise après l'encodage.

Bonus:
  • Permet d'encoder le nom des joueurs
  • Permet d'avoir un nombre de joueurs variables (minimum 2)
